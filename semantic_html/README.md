# 📚 Concepts Used

| Tag | Purpose |
|---|---|
| `<header>` | Top section of webpage |
| `<nav>` | Navigation links |
| `<main>` | Main webpage content |
| `<section>` | Group related content |
| `<article>` | Independent content block |
| `<footer>` | Bottom section of webpage |

---

# 💻 HTML Code

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Semantic HTML</title>
</head>
<body>

    <header>
        <h1>My Website</h1>
    </header>

    <nav>
        <a href="#">Home</a> |
        <a href="#">About</a> |
        <a href="#">Contact</a>
    </nav>

    <main>

        <section>
            <h2>About HTML</h2>

            <p>
                Semantic HTML improves webpage structure and readability.
            </p>
        </section>

        <article>
            <h2>Latest Article</h2>

            <p>
                HTML5 introduced semantic elements for better webpage organization.
            </p>
        </article>

    </main>

    <footer>
        <p>© 2026 My Website</p>
    </footer>

</body>
</html>
```
