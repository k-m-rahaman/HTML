# 📚 Concepts Used

| Tag | Purpose |
|---|---|
| `<h1>` to `<h6>` | Create headings of different sizes |
| `<p>` | Define paragraphs |
| `<br>` | Insert line break |
| `<hr>` | Create horizontal line |
| `<!-- -->` | Add comments in HTML |

---

# 💻 HTML Code

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Headings and Paragraphs</title>
</head>
<body>

    <h1>Main Heading</h1>
    <h2>Sub Heading</h2>
    <h3>Smaller Heading</h3>

    <p>
        HTML is used to create webpages.
    </p>

    <hr>

    <p>
        This is another paragraph.<br>
        This line is after break tag.
    </p>

</body>
</html>
```
