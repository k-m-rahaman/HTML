# 📚 Concepts Used

| Tag | Purpose |
|---|---|
| `<ol>` | Create ordered list |
| `<ul>` | Create unordered list |
| `<li>` | Define list items |
| `<table>` | Create table |
| `<tr>` | Define table row |
| `<th>` | Define table heading |
| `<td>` | Define table data |

---

# 💻 HTML Code

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Lists and Tables</title>
</head>
<body>

    <h1>Ordered List</h1>

    <ol>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ol>

    <hr>

    <h1>Unordered List</h1>

    <ul>
        <li>Apple</li>
        <li>Mango</li>
        <li>Orange</li>
    </ul>

    <hr>

    <h1>Student Table</h1>

    <table border="1">

        <tr>
            <th>Name</th>
            <th>Marks</th>
            <th>Grade</th>
        </tr>

        <tr>
            <td>Kazi</td>
            <td>90</td>
            <td>A</td>
        </tr>

        <tr>
            <td>Rahim</td>
            <td>85</td>
            <td>B</td>
        </tr>

    </table>

</body>
</html>
```
