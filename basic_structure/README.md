# 📚 Concepts Used

| Tag | Purpose |
|---|---|
| `<!DOCTYPE html>` | Defines HTML5 document |
| `<html>` | Root element of webpage |
| `<head>` | Stores webpage metadata |
| `<title>` | Sets browser tab title |
| `<body>` | Contains webpage content |
| `<meta>` | Defines character encoding & responsiveness |

---

# 💻 HTML Code

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic Structure</title>
</head>
<body>

    <h1>Welcome to HTML</h1>
    <p>This is my first HTML webpage.</p>

</body>
</html>
```
